# MMM-M
Current Movies module for MagicMirror 2


In terminal window:

~MagicMirror/modules

git clone https://github.com/cowboysdude/MMM-M

~MagicMirror/modules/MMM-M

npm install

____________________________________________________

config.js entry:

       {
		  	module: 'MMM-M',
		  	position: 'top_right',
		  },
