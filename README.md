# MMM-M
Current Movies module for MagicMirror 2


List of current movies out now...

How to install:

~MagicMirror/modules

git clone https://github.com/cowboysdude/MMM-M

~MagicMirror/modules/MMM-M

npm install


---------------------------Config.js entry---------------------------

      {
	      	disabled: false,
		  	  module: 'MMM-M',
		  	  position: 'top_center',
		   },
